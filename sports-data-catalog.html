🏆 Sports Data Catalog

A guide to available scrapers and datasets for sports data visualization projects

27+ scrapers / scripts 9 sports covered 190+ data files Sources: sports-reference.com · baseball-reference.com · stats.ncaa.org · hockey-reference.com · pro-football-reference.com · fbref.com · pgatour.com · transfermarkt.com Language: R (tidyverse, rvest)
College Football NFL College Basketball NBA MLB NCAA Baseball NCAA Softball NCAA Volleyball NHL Soccer Other Sports
How to use this catalog Each section lists the scraper scripts available (left panel) and the data files already collected (right panel). Scrapers can be re-run to update data. All scripts are R Markdown (.Rmd) or plain R (.R) files using the tidyverse and rvest packages unless otherwise noted.

Scrapers
CFBLogsScrape.Rmd
Scrapes FBS team game-by-game offensive and defensive logs for a single season. Edit the year variable to target any season.
sports-reference.com/cfb HTML table scraping → footballlogs{year}.csv
CFBLogsScrapeOlderYears.Rmd
Same as above but configured for 2020 season — useful template for scraping pre-2022 seasons where conference structures differed.
sports-reference.com/cfb HTML table scraping → footballlogs{year}.csv

Scrapes all-time win-loss records, bowl records, and historical program data for every FBS program.

Scrapes FBS quarterback passing statistics for a given season (minimum attempts filter applied). Update the season variable.

Scrapes full team rosters (player name, position, height, weight, year) for all FBS teams from stats.ncaa.org. Requires a seed CSV of team URLs (football2020.csv).
