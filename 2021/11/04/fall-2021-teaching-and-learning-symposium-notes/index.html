Fall 2021 Teaching and Learning Symposium Notes

2021-11-04

What you’ll need:

  • R 4.1.2 for your operating system
  • R Studio
  • A Github account
  • Github desktop

What you’ll do:

  1. Install everything.
  2. Open R studio
  3. In the console, type install.packages("bookdown")
  4. File > New Project
  5. New Directory > Book Project with Blogdown
  6. Pick a name, a location and a style of book.
  7. Write your book.

Publishing your book

  1. Log into Github and create a new repository called yourusername.github.io where yourusername is … your username. For example, mine is mattwaite.github.io
  2. In Github Desktop, log into your github account and clone your username.github.io repository. It’s better if you don’t put it in the same folder as your book.
  3. In the folder created when you cloned your repository, create a new folder that is yourbookname, where yourbookname is … you get it. Whatever you call this is going to be your book url – yourusername.github.io/yourbookname.
  4. In R Studio, go to the Build tab and click Build Book.
  5. Copy the contents of the _book folder.
  6. Paste those contents into the yourusername.github.io/yourbookname
  7. Open Github Desktop. You should see changes to commit.
  8. Fill in the short summary – think of it as a note about what these changes are so you can keep track of which does what.
  9. Click Commit to Main.
  10. Click Push Origin.
  11. Wait a few minutes.
  12. Visit your url.

Want more? There’s a book … in bookdown, of course.
